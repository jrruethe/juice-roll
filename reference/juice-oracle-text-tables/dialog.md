# Dialog

**1d10: Direction & Tone; 1d10: Subject; Doubles: End; *Past/Present***

This is a 2D lookup table where one die determines the row (Direction/Tone) and another die determines the column (Subject).

## Direction & Tone (Row)

| Roll | Direction | Tone |
|------|-----------|------|
| 1-2 | Neutral | Them |
| 3-5 | Defensive | Me |
| 6-8 | Aggressive | You |
| 9-0 | Helpful | Us |

## Subject (Column)

| Roll | 1-2 | 3-5 | 6-8 | 9-0 |
|------|-----|-----|-----|-----|
| | Neutral/Them | Defensive/Me | Aggressive/You | Helpful/Us |

## Dialog Matrix

|  | 1-2 Neutral/Them | 3-5 Defensive/Me | 6-8 Aggressive/You | 9-0 Helpful/Us |
|--|------------------|------------------|---------------------|----------------|
| **1-2 Neutral/Them** | *Fact* | *Denial* | Query | *Denial* | Action |
| **3-5 Defensive/Me** | *Want* | *Query* | *Need* | *Query* | *Fact* |
| **6-8 Aggressive/You** | Action | Need | **Fact** | Action | Denial |
| **9-0 Helpful/Us** | Need | Query | *Denial* | Query | Want |
|  | Query | Support | Query | Support | Need |
